I love me a good podcast, but it's hard not to notice how criminally generic most podcast sites tend to be. This site is a functional attempt at creating a unique look for a podcast site. It was created using the SASS preprocessor, and was built ontop of a custom wordpress theme. The user can make a post using the wordpress dashboard, and episodes can be linked through the soundcloud API for direct play on the site.
